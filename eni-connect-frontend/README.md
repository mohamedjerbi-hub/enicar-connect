# ENI Connect - Plateforme Web Communautaire

Plateforme web centralisée pour l'École Nationale d'Ingénieurs de Carthage (ENI Carthage).

## Description

ENICAR Connect est une plateforme communautaire qui centralise toutes les communications et services de l'école sur une seule interface. Elle remplace les canaux de communication dispersés (WhatsApp, Facebook) et digitalise les procédures administratives.

## Fonctionnalités

### Module Réseau Social Interne
- Fil d'actualités avec publications, likes, commentaires
- Groupes par filière, promotion, clubs
- Événements et calendrier
- Messagerie instantanée

### Module Réseau Professionnel
- Offres d'emploi et stages
- Profils d'entreprises partenaires
- Annuaire des alumni

### Module Services Utiles
- Formulaires administratifs digitalisés
- Gestion de documents
- Notifications en temps réel

## Installation

```bash
# Installer les dépendances
npm install

# Lancer le serveur de développement
npm start
```

L'application sera accessible sur `http://localhost:4200/`

## Technologies

- Angular 17
- TypeScript
- CSS3
- RxJS
